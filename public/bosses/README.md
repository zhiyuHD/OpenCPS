# 📦 出厂默认 Boss 图片库

## 🎯 用途说明

这个目录存放的是**出厂默认 Boss 图片**，会被打包进 EXE 安装包。

用户安装软件后，如果没有上传自定义图片，系统会优先显示这里的默认图片。

---

## 📂 图片放置规则

将处理好的 Boss 图片直接放入此目录：

- 文件名必须与 Boss 名称完全一致
- 支持格式：`.jpg`、`.png`、`.jpeg`
- 建议尺寸：512x512 或更高
- 风格：建议使用威严、霸气的 Boss 形象

**示例**：
```
public/bosses/
├── 火域领主.jpg
├── 水域霸主.png
├── 暗影魔王.jpg
├── 雷霆之神.png
└── ...
```

---

## 🔄 图片读取优先级

系统采用四级降级机制：

1. **用户自定义图片**（存储在 `%AppData%/学习王国：李翔老师精灵系统/bosses/`）
2. **出厂默认图片**（此目录，打包进 EXE）
3. **数据库 Base64 图片**（IndexedDB）
4. **默认巨型 Emoji 占位符**

---

## 💡 使用场景

- 开发者：在此目录放置默认 Boss 图片，打包进 EXE
- 用户：通过系统设置上传自定义 Boss 图片，保存到 userData 目录
