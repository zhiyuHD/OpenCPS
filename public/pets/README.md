# 📦 出厂默认精灵图片库

## 🎯 用途说明

这个目录存放的是**出厂默认图片**，会被打包进 EXE 安装包。

用户安装软件后，如果没有上传自定义图片，系统会优先显示这里的默认图片。

---

## 📂 图片放置规则

将处理好的精灵图片直接放入此目录：

- 文件名必须与精灵名称完全一致
- 支持格式：`.jpg`、`.png`、`.jpeg`
- 建议尺寸：512x512 或更高

**示例**：
```
public/pets/
├── 金刚石.jpg
├── 水晶.png
├── 火焰鸟.jpg
├── 雷电兽.png
└── ...
```

---

## 🔄 图片读取优先级

系统采用四级降级机制：

1. **用户自定义图片**（存储在 `%AppData%/学习王国：李翔老师精灵系统/pets/`）
2. **出厂默认图片**（此目录，打包进 EXE）
3. **数据库 Base64 图片**（IndexedDB）
4. **默认 Emoji 占位符**

---

## 💡 使用场景

- 开发者：在此目录放置默认图片，打包进 EXE
- 用户：通过系统设置上传自定义图片，保存到 userData 目录
