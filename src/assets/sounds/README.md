# 🎵 音效音频文件夹

## 文件夹结构

```
src/assets/sounds/
├── bgm/              # 背景音乐
│   ├── happy.mp3     # 欢快背景音乐
│   ├── calm.mp3      # 平静背景音乐
│   └── exciting.mp3  # 激动背景音乐
│
├── effects/          # 音效
│   ├── add-points.mp3      # 加分音效
│   ├── deduct-points.mp3   # 扣分音效
│   ├── level-up.mp3        # 升级音效
│   ├── feed.mp3            # 喂食音效
│   ├── pet.mp3             # 抚摸音效
│   ├── click.mp3           # 点击音效
│   ├── success.mp3         # 成功音效
│   ├── fail.mp3            # 失败音效
│   ├── heart.mp3           # 爱心特效音效
│   ├── freeze.mp3          # 冰冻音效
│   └── sigh.mp3            # 叹气音效
│
└── README.md         # 本文件

```

## 音频格式要求

- 格式：MP3 或 WAV
- 大小：建议每个文件不超过 500KB
- 时长：
  - 背景音乐：1-3分钟（可循环）
  - 音效：0.5-2秒

## 使用说明

1. 将音频文件放入对应文件夹
2. 文件名必须与上述列表一致
3. 系统会自动加载这些音频
4. 如果文件不存在，系统会静默跳过

## 推荐音源

- 免费音效网站：
  - Freesound.org
  - Zapsplat.com
  - Mixkit.co
  
- 背景音乐：
  - YouTube Audio Library
  - Incompetech.com
  - Bensound.com

## 注意事项

- 请确保音频文件有合法使用权
- 音量建议统一调整到适中水平
- 背景音乐建议选择无版权音乐
